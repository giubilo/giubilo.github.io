http://docs.alfresco.com/4.2/topic/com.alfresco.enterprise.doc/pra/1/concepts/cmis-1.1-browser-binding.html The Browser Binding

In addition to the existing XML-based AtomPub and Web services bindings, CMIS 1.1 provides a simpler JSON-based binding. The browser binding is designed for web applications and is easy to use just with HTML and JavaScript. It uses just two verbs, GET and POST, and resources are referenced using simple and predictable URLs. You reference content in the repository by using the two URLs returned by the getRepositories or getRepositoryInfo service: rootFolderUrl repositoryUrl Objects can then be referenced in two ways; by their Id ?objectId= by their path /
